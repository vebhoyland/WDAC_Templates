# WDAC Templates
WDAC XML templates for building Multi Policy WDAC .cip policies or as a starting point for new users.

One simple base policy and one with deny for lolbins.
Warning! Lolbins change over time. If you want the newest Microsoft recommended lolbin blocks, please reffer to an updates source or add your own custom limitations.

Blogpost: https://hoyland.cloud/building-wdac-policies-with-wdac-wizard/
